# i18n
i18n.js

## Usage
```
 <script type="text/javascript">
     I18n = {}
     I18n.options = {locale: "en"};
 </script>
 
  load the file: i18n.js
  load translate js
 
 <script type="text/javascript">
     I18n.T("hello", {name: "you"})
     //"你好, you "
     I18n.T("helloObj.test")
     //"just a example"
 </script>
 ```
